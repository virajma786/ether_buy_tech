Overview
Ether Buy Tech Market is an e-commerce platform that enables secure and decentralized transactions for tech products. It provides a seamless online shopping experience while integrating blockchain technology for transparency and security.

Tech Stack
Frontend: React, HTML, CSS, Bootstrap
Backend: Node.js, Express.js
Database: MySQL
Blockchain Integration: Ethereum, MetaMask
Features
✔️ User Authentication – Secure login & signup
✔️ Decentralized Transactions – Ethereum-based payments
✔️ Product Listing & Management – Browse and manage tech products
✔️ Cart & Checkout System – Add to cart and proceed with payments
✔️ Blockchain Security – Transaction history on the Ethereum blockchain
✔️ Admin Panel – Manage users, products, and orders
